# Toters Iraq — Customer Sentiment Analysis 🍊📊

## Overview
Analyzed 449 real customer reviews scraped from Google Play Store (Iraq region) to understand user satisfaction and identify key pain points.

## Key Findings
- 54% Negative | 41% Positive | 4% Neutral
- Main complaint: Login & account issues
- Second complaint: Slow delivery
- Third complaint: App crashes & bugs

## Business Recommendations
1. Fix OTP/login system → add Google Sign-in
2. Improve real-time delivery tracking
3. Prioritize app stability before marketing expansion

## Tools Used
Python | Pandas | Google Play Scraper | Matplotlib | WordCloud

## Data Source
Google Play Store — Public reviews (Iraq, Arabic)
